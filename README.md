# dotnet-react-material
.NET Core template with ReactJS &amp; Material UI

# Installation
```sh
dotnet new -i RamosISW.ReactMaterialTemplate.CSharp
```

# Usage

```sh
dotnet new react-material
```

Creates an outpur dir like this
```
              
├───ClientApp
│   │   .gitignore
│   │   package-lock.json
│   │   package.json
│   │   README.md
│   │               
│   ├───public
│   │       favicon.ico
│   │       index.html
│   │       manifest.json
│   │       
│   └───src
│           App.css
│           App.js
│           App.test.js
│           index.css

│

```
